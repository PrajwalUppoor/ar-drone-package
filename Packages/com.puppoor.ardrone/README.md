# AR Drone Package

Plug-and-play AR drone controller package for Unity.

## Installation

Via Package Manager:

Window → Package Manager → + → Add package from Git URL

Paste:

https://github.com/PrajwalUppoor/ar-drone-package.git?path=/Packages/com.puppoor.ardrone

## Contents

- DroneController
- DroneInput
- ARPlacement
- IndustrialMachine

## Usage

Attach DroneController to your drone prefab.
Use ARPlacement for plane placement.
Feed inputs via DroneInput.

Done.
